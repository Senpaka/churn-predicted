## 📊 Метрики

| Метрика   | Значение  |
|:----------|:----------|
| Recall    | 68.7%     |
| Precision | 46.6%     |
| F1-Score  | 55.6%     |
| Accuracy  | 77.6%     |

## 🎯 Бизнес-задача

Нам важнее удержать клиента, даже ценой лишних звонков. Модель позволяет:
- Найти **68.7%** уходящих клиентов
- При этом **46.6%** обзвоненных действительно уходят
- Остальные лишние звонки — приемлемая плата за высокий охват

## 📈 SHAP анализ

| Признак             | shap значение | Значение                                                  |
|:--------------------|:--------------|:----------------------------------------------------------|
| **NumOfProducts**   | 0.900607      | Клиенты с 3+ продуктами уходят чаще                       |
| **Age_Group_50-64** | 0.525364      | Говорит что люди в пожилом возрасте довольно часто уходят |
| **IsActiveMember**  | 0.479360      | Неактивный = высокий риск ухода                           |
| **Balance**         | 0.311496      | Люди с высоким балансом чаще уходят                       |
| **Gender_Male**     | 0.288431      | Мужчины немного чаще уходят                               |


## Готовая API

### Эндпоинты

| Метод  | Эндпоинт               | Описание                             |
|:-------|:-----------------------|:-------------------------------------|
| GET    | `/`                    | Информация о сервисе                 |
| GET    | `/health`              | Проверка работоспособности           |
| POST   | `/prediction`          | Предсказание для одного клиента      |
| POST   | `/prediction_batch`    | Предсказание для нескольких клиентов |
| POST   | `/predict_proba`       | Только вероятность оттока            |
| POST   | `/prediction_from_csv` | Предсказание из CSV файла            |



## 🚀 Быстрый старт

### Установка

#### 1. Клонирование репозитория
```bash
 git clone https://github.com/yourusername/churn-prediction.git
 cd churn-prediction
```
#### 2. Установка зависимостей
```bash
 #Создание виртуального окружения
 python -m venv .venv
 source .venv/bin/activate # linux/mac
 # .venv\Scripts\activate   # Windows

 # Установка зависимостей
 pip install -r requirements.txt
```

#### 3. Обучение модели
```bash
 python scripts/train_pipeline.py
```

#### 4. Запуск API
```bash
 uvicorn api.app:app --reload --host 0.0.0.0 --port 8000
```
