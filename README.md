# Churn Prediction Model

Модель для предсказания оттока клиентов банка.

## 📊 Метрики

| Метрика | Значение |
|---------|----------|
| Recall | 68.7%    |
| Precision | 46.6%    |
| F1-Score | 55.6%    |
| Accuracy | 77.6%    |

## 🎯 Бизнес-задача

Нам важнее удержать клиента, даже ценой лишних звонков. Модель позволяет:
- Найти **68.7%** уходящих клиентов
- При этом **46.6%** обзвоненных действительно уходят
- Остальные лишние звонки — приемлемая плата за высокий охват

## 📈 Пример для 1000 клиентов

| Показатель | Значение |
|------------|----------|
| Всего клиентов | 1,000 |
| Реально уйдут | 204 (20.4%) |
| **Найдено уходящих (TP)** | **140** |
| Пропущено уходящих (FN) | 64 |
| Ложных тревог (FP) | 160 |
| Правильно оставшихся (TN) | 636 |
